# Contributing to ConsideredWeb

We welcome contributions! Whether it’s a bug report, feature suggestion, or pull request — thank you for helping improve ConsideredWeb.

## How to Contribute

1. **Fork** the repository and create your branch from `main`.
2. **Make your changes**:
   - Keep code style consistent.
   - Add/update tests where relevant.
3. **Commit** with clear messages.
4. **Open a Pull Request** describing your changes and linking any relevant issues.

## Code of Conduct
This project follows a [Code of Conduct](./CODE_OF_CONDUCT.md). By participating, you agree to uphold it.

## Feedback & Usage
If you’re using ConsideredWeb in a project, we’d love to hear about it!  
Please [open an issue](../../issues) or share feedback so we can keep improving.
